# eos

This repository provides C++, python, and MATLAB packages for cubic equations of state (EoS).

# Development Status

## C++

- Package for pure component systems
  - EoS
    - [x] Van der Waals EoS
    - [x] Soave-Redlich-Kwong EoS
    - [x] Peng-Robinson EoS
  - Thermodynamic property functions
    - [x] Pressure
    - [x] Z-factor
    - [x] Fugacity coefficients
    - [ ] Enthalpy
    - [ ] Entropy
    - [ ] Helmholtz energy
  - Phase equilibria
    - [ ] Vapor pressure
- Package for multi component systems
  - EoS
    - [ ] Van der Waals EoS
    - [ ] Soave-Redlich-Kwong EoS
    - [ ] Peng-Robinson EoS
  - Thermodynamic property functions
    - [ ] Pressure
    - [ ] Z-factor
    - [ ] Fugacity coefficients
    - [ ] Enthalpy
    - [ ] Entropy
    - [ ] Helmholtz energy
  - Phase equilibria
    - [ ] Bubble point pressure
    - [ ] Dew point pressure
    - [ ] Vapor-Liquid flash
